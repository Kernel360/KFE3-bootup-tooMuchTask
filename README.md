# tooMuchTask 🎯

경매 시스템 프로젝트

## 🛠️ Tech Stack

- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Build System**: Turborepo (Monorepo)

## 📁 Project Structure

```
tooMuchTask/
├── apps/
│   └── web/          # 메인 웹 애플리케이션
├── packages/
│   └── ui/           # 공통 UI 컴포넌트
└── turbo.json
```

## 🚀 Getting Started

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```
